## 🛠️ Areas of Expertise

We build search-agent-friendly and ultra-fast web architectures optimized for both humans and modern AI search engines.

*   **Technical SEO & Schema Markup**: Custom JSON-LD schema design, semantic HTML, and indexability tuning.
*   **GEO & AEO (AI Search Optimization)**: Strategy and markup optimization for LLM-driven answer engines (e.g., ChatGPT, Gemini, Perplexity, Naver Cue:).
*   **Web & CMS Development**: High-performance custom websites on WordPress (WooCommerce), Cafe24, and modern headless frameworks (Astro, Next.js).
*   **Core Web Vitals Optimization**: Performance tuning targeting near 100/100 PageSpeed Insights scores.
